Ubuntu

A distro Ubuntu é um sistema operacional Linux popular, de código aberto, baseado no Debian, conhecido por sua facilidade de uso, estabilidade e vasto suporte, sendo ideal para iniciantes e servidores, com foco em software livre e uma vasta comunidade, oferecendo versões LTS (Long-Term Support) e "Flavors" (Kubuntu, Lubuntu, etc.) com diferentes ambientes gráficos.

Arch

Arch Linux é uma distribuição GNU/Linux desenvolvida independentemente, de propósito geral e com um modelo de lançamento contínuo (rolling release), conhecida por sua simplicidade, modernidade, pragmatismo e centralidade no usuário (a atitude "faça você mesmo" ou "DIY"). Ela fornece um sistema base mínimo, incentivando os usuários a configurá-lo do zero para atender às suas necessidades específicas.

Kali-Linux

Kali Linux é uma distribuição GNU/Linux baseada em Debian, voltada para testes de penetração, hacking ético, análise forense digital e engenharia reversa. Ele vem pré-instalado com centenas de ferramentas para essas atividades, sendo mantido pela Offensive Security. O sistema é gratuito, de código aberto e pode ser usado a partir de um Live CD/USB ou instalado como sistema principal.

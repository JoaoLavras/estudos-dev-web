Lista

Uma lista é uma estrutura de dados que armazena elementos em sequência, permitindo acessar, inserir ou remover itens em qualquer posição. É flexível e muito usada para organizar conjuntos variados de informações.

Pilha

A pilha funciona pelo princípio LIFO, onde o último elemento inserido é o primeiro a sair. É como uma pilha de livros: você só consegue pegar ou colocar algo no topo.

Fila

A fila segue o princípio FIFO, em que o primeiro elemento a entrar é o primeiro a sair. Funciona como uma fila de pessoas esperando atendimento, mantendo a ordem de chegada.

Algoritmos eficientes são aqueles que resolvem um problema usando poucos recursos — principalmente tempo e memória — mesmo quando a quantidade de dados cresce, como ocorre em algoritmos de busca binária ou ordenação por merge sort. Já os algoritmos não eficientes tendem a consumir muito mais tempo ou espaço, tornando-se lentos e impráticos para grandes volumes de informação, como o bubble sort ou buscas exaustivas.
